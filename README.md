# Customer Segmentation & Buying Behavior Dashboard

## Project Overview

This project is an interactive Tableau dashboard designed to analyze customer segmentation, purchasing behavior, revenue trends, loyalty patterns, and customer satisfaction insights.

The dashboard helps businesses and management teams make data-driven decisions by understanding customer spending patterns, identifying high-value customers, and improving marketing strategies.

---

# Business Objective

The main objective of this project is to:

* Analyze customer buying behavior
* Identify profitable customer segments
* Monitor customer loyalty and retention
* Track revenue trends and product performance
* Improve business decision-making using analytics
